Built using Next.js, this web app enables users to locate near-by public wifi in NYC. 

## Features
- Geolocation
- Google Map
- Public Wifi Locator

## Usage
Clone the repository. Run the app using:
```bash
npm run dev
```

## Tests
Prerequisite: Playwright

Run the tests using:
```bash
npx playwright test
```
